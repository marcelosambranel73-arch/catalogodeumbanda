# Catálogo do Curso de Umbanda

## Projeto
Site estático com material de estudo sobre Umbanda brasileira.

## Estrutura
```
├── index.html              # Página principal
├── Gemini_Generated_Image_lrhrvhlrhrvhlrhr.png  # Imagem de fundo
├── database.json          # Banco de dados de documentos
├── extrair_doc.py        # Script original de extração
├── 01-Fundamentos/       # 3 documentos
├── 02-Orixas/            # 5 documentos + subpastas
├── 03-Entidades/         # 11 documentos
├── 04-Hierarquia-Organizacao/  # 2 documentos
└── 05-Praticas-Rituais/ # 13 documentos
```

## Funcionalidades
- Página inicial com introdução sobre Umbanda
- Catálogo com 47 documentos organizados por categoria
- Busca rápida
- Leitor inline (sem abrir nova aba)
- Navegação entre documentos (anterior/próximo)
- Design responsivo com tema escuro

## Como usar
Abra `index.html` em um navegador web.
